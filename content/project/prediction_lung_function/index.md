The core objective of this modeling effort is to develop predictive tools using deep learning for the precise quantification and longitudinal forecasting of interstitial lung disease (ILD) progression in systemic sclerosis (sSc) patients. This enables earlier detection of patients facing rapid lung function decline for timely therapeutic decisions. (read more here: [link](https://www.medrxiv.org/content/10.1101/2024.04.25.24306365v1))
